# Café Éphémère

Projet **CWAD** — *Communication Web Agence De-voyage*  
Projet universitaire IUT — Site web d'un café thématique immersif à Paris.

## Présentation

Le **Café Éphémère** est un café unique situé au cœur du Marais (Paris 3e) où chaque mois, un nouveau thème transporte les visiteurs dans des univers variés et immersifs : Steampunk Explorers, Jardin Enchanté, Cyberpunk Metropolis...

## Pages du site

- **Accueil** — Présentation du concept du Café Éphémère
- **Activités** — Les thèmes et expériences proposés
- **Contact** — Formulaire de réservation, coordonnées et carte

## Technologies

- HTML5 / CSS3
- Design responsive
- Vidéo de fond YouTube intégrée
